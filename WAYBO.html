WAY-BO

Line following robot with ultrasonic sensor for obstacle avoiding. (Inspired by industrial robots for supplying materials and automating workflow) The event consists of two rounds. First round will be a preliminary round were teams are given MCQ on robotics. The top 5 teams will be selected for the final round. In the final round participants are given components like sensors, motors and a development board (Arduino, participants can code using python or C ). Using the given components they have to build a robot that has to execute a task given by the judges. The task may be like solving a maze (with obstacles and twisted lines) or avoiding obstacles by following the line and reaching the destination.
